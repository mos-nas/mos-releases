# Security Policy

## Reporting a Vulnerability

Please **do not** report security vulnerabilities through public GitHub issues.

Instead, please email:
**admin@minenet.at**

As this is an Open Source project maintained by an individual, responses and fixes are provided on a **best-effort basis**. I'll do my best to respond as soon as possible.

## When Reporting

Please include (if possible):
- Description of the issue and potential impact
- Steps to reproduce / proof of concept
- Affected versions / commits
- Any relevant logs or configuration details

## What to Expect

- I will acknowledge your report
- I will investigate and work on a fix
- I will keep you informed about the progress
- Credit will be given for responsible disclosure (if desired)
